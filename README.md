# The Super App

React + Vite frontend assignment implementation with registration, category selection, dashboard widgets, notes persistence, timer controls, and movie discovery.

## Run

```bash
npm install
npm run dev
```

## Optional API keys

Set these in a local `.env` file if you want live data instead of fallback content:

- `VITE_OPENWEATHER_API_KEY`
- `VITE_NEWS_API_KEY`
- `VITE_TMDB_API_KEY`
